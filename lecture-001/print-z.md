# Print Z

> 1. You are required to print a 'z' of size 5 using '*'.

```text
Input Format
There is no input
Output Format
print-z-in-java

Constraints
No Constraints
Sample Output

*****
   *
  *
 *
*****
```

![Screenshot 2022-01-11 133809](https://user-images.githubusercontent.com/64803628/148904470-4edd2e9b-9ec8-466c-abc9-03d119f8dd4e.png)


## Solution

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("*****");
        System.out.println("   *");
        System.out.println("  *");
        System.out.println(" *");
        System.out.println("*****");
    }
}
```
